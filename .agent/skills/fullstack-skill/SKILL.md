# Fullstack Development Skill

Capacidades integradas para gerenciar frontend React e backend Python/FastAPI.

## Especialidades
- **Frontend Moderno**: Implementação de UI/UX com Tailwind, Framer Motion e React Query.
- **Python Backend/AI**: Automação, APIs assíncronas com FastAPI e scriptagem eficiente via Uvicorn.

## Comandos Rápidos
- `start`: Inicia o ambiente fullstack unificado via orquestrador (PM2).
- `dev`: Inicia instâncias do dev server e reloaders.

## Padrões de Código
- Sempre use tipagem forte (TypeScript para Node, TypeHints para Python).
- Siga as regras globais definidas em `.agent/rules/global.md`.
